# Screenshots

Drop PNG screenshots here with these exact filenames (referenced in the main README):

| File | Suggested content |
|---|---|
| `chat.png` | Telegram chat with the bot — a few back-and-forth messages |
| `news-digest.png` | The morning AI news digest |
| `calendar-gmail.png` | Bot listing calendar events or summarizing emails |
| `image-gen.png` | Bot generating or editing an image |
| `usage.png` | `/usage` and `/memories` output |
| `voice.png` | A voice reply (or the waveform UI from Telegram) |

Tips:
- Crop to just the bot conversation area (no system bars, no other chats)
- Use Telegram's dark or light mode consistently
- ~800–1200px wide is plenty
